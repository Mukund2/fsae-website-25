# Lessons

## Design System
- **McLaren-inspired does NOT mean dark theme everywhere.** McLaren Racing's homepage uses dark hero but light/white body sections. Only specific CTAs and footer are dark. Default body sections to `bg-background` (warm cream), not `bg-[#111]`.
- **Accent colors are GOLD (#B8965A) and BLUE (#0055A2)** for the SJSU Spartan Racing theme. Never use orange.
